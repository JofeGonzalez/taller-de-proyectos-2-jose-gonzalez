Contributing
=============

Gracias por contribuir al proyecto. Estas notas sirven para que puedas probar, modificar o extender la app.

1) Configuración local

```powershell
npm install
```

2) Ejecutar tests

```powershell
npm test
```

3) Flujo de trabajo básico

- Crea una rama por característica: `git checkout -b feature/nombre`.
- Realiza cambios en pequeñas unidades con mensajes claros: `git commit -m "feat: añadir export CSV"`.
- Abre un pull request para revisión.

4) Sugerencias para contribuciones

- Añade tests para cualquier comportamiento nuevo.
- Mantén la UI accesible (labels, roles y `aria-*` donde aplican).
- Si introduces dependencias externas, documenta la razón en el `README.md`.

5) Contacto

Si necesitas ayuda, deja un comentario en el PR o abre un issue.
